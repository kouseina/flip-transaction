<h1>Flip Transaction</h1>
<h2>How to use?</h2>
<h4>1. Download or clone this repo.</br></h4>
<h4>2. Install dependencies.</br></h4>

```
npm install
// or
yarn install
```

<h4>3. Run project on iOS / Android.</h4>

```
react-native run-android // npm run android
// or
react-native run-ios // yarn android
```
